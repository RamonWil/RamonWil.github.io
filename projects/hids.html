Host-Based Intrusion Detection System (HIDS)

Python-based monitoring system designed to detect unauthorized file modifications and maintain system integrity.

System Architecture

The HIDS monitors critical directories and system logs. It uses a modular Python framework that applies cryptographic hashing (SHA-256) to detect unauthorized changes in real time.

Detailed Logging & Alerts

Each detection event is recorded with timestamps, file paths, and change details. Alerts are logged and optionally sent via email or console output for quick administrator review.

Custom Configuration

Users can define watch paths, frequency intervals, and exclusion rules via a configuration file, enabling flexibility for system administrators and developers.
