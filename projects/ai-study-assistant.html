AI-Powered Study Assistant

A full-stack AI web app that transforms uploaded notes into summaries, flashcards, and quizzes using OpenAI’s API — deployed on Azure with Docker and SQL integration.

Smart Upload & Data Parsing

Users can upload lecture notes or PDFs, which the backend processes and segments intelligently using natural language understanding and vector embeddings for context. This ensures concise, relevant study material generation.

Automated Flashcard & Quiz Creation

The app generates structured flashcards and practice questions using OpenAI's GPT models, storing data in SQL while providing a clean, interactive UI for studying.

Interactive Practice Mode

Users can test their knowledge through adaptive quizzes with instant feedback, powered by Flask and dynamic JavaScript components for real-time performance tracking.
